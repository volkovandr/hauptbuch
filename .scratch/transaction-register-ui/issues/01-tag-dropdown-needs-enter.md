# Tag selection from dropdown still requires a second Enter

Status: needs-triage
Severity: minor
Area: Transaction register

When selecting a Tag from the dropdown, you have to press Enter again after selecting,
because right after the selection the field is still in the "editing" state.

Preferred behaviour: picking a Tag from the dropdown (by mouse or keyboard) should select
and confirm it immediately. Enter should only be necessary when typing a tag name by hand.
